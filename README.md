# Dari

Dari Modern Online Courses.

### Documentation

https://dari-7f111.web.app/

### Getting Started

The steps to compile and get started with development are covered in detail in documentation mentioned above, but the summary is:

- npm install -g gulp-cli
- npm install
- gulp
